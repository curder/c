# 注释的使用

---

## 单行注释

- `//` 进行单行注释
    + 使用范围:任何地方都可以写注释：函数外面、里面，每一条语句后面
    + 作用范围: 从第二个斜线到这一行末尾
    + 快捷键:`Command+/`


## 多行注释
- `/* */` 进行多行注释
     + 使用范围:任何地方都可以写注释：函数外面、里面，每一条语句后面
     + 作用范围: 从第一个`/*到最近的一个*/`


## 注释使用注意
- 1.单行注释可以嵌套单行注释、多行注释

```
// test1// test2
// /* test3 */
// test4
```

- 2.多行注释可以嵌套单行注释

```
/*
// 第一行注释
// 描述：第一个C语言程序作用：这是一个主函数，C程序的入口点
 */
 ```
- 3.多行注释不能嵌套多行注释

```
/* 哈哈
     /*嘻嘻*/
 呵呵 */
 ```
